

# Novo Campo: Unidade Base

## Resumo

Criar uma nova tabela auxiliar `unidades_base` e vincula-la aos clientes via `unidade_base_id`. Adicionar o campo no formulario do cliente, na listagem principal e substituir o filtro rapido de recorrencia pelo filtro de Unidade Base.

---

## Etapa 1: Migration SQL

```sql
-- Criar tabela unidades_base
CREATE TABLE public.unidades_base (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome text NOT NULL
);

ALTER TABLE unidades_base ENABLE ROW LEVEL SECURITY;
CREATE POLICY "auth_read_unidades_base" ON unidades_base FOR SELECT TO authenticated USING (true);
CREATE POLICY "auth_write_unidades_base" ON unidades_base FOR ALL TO authenticated USING (true) WITH CHECK (true);

-- Adicionar coluna em clientes
ALTER TABLE public.clientes ADD COLUMN unidade_base_id bigint REFERENCES unidades_base(id);
CREATE INDEX idx_clientes_unidade_base ON public.clientes (unidade_base_id);
```
