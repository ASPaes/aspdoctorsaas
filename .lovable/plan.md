Remover as colunas `codigo_fornecedor`, `link_portal` e `fornecedor_id` da tabela `produtos`, deixando apenas `id` e `nome`.

ALTER TABLE produtos DROP COLUMN IF EXISTS link_portal;
ALTER TABLE produtos DROP COLUMN IF EXISTS fornecedor_id;

## 2. Criar tabela Origens de Venda

Nova tabela auxiliar `origens_venda` para substituir o campo texto livre `origem_venda` nos clientes.

### Banco de dados
```sql
-- Criar tabela
CREATE TABLE public.origens_venda (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome text NOT NULL
);
ALTER TABLE origens_venda ENABLE ROW LEVEL SECURITY;
CREATE POLICY "auth_read_origens_venda" ON origens_venda FOR SELECT TO authenticated USING (true);
CREATE POLICY "auth_write_origens_venda" ON origens_venda FOR ALL TO authenticated USING (true) WITH CHECK (true);

-- Trocar coluna em clientes: texto -> FK
ALTER TABLE clientes DROP COLUMN IF EXISTS origem_venda;
ALTER TABLE clientes ADD COLUMN origem_venda_id bigint REFERENCES origens_venda(id);
CREATE INDEX idx_clientes_origem_venda ON clientes(origem_venda_id);
```

Tambem atualizar a view `vw_clientes_financeiro` para trocar `origem_venda` (text) por `origem_venda_id` (bigint).
